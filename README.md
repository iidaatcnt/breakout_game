# ブロック崩しゲーム

このゲームは、古典的なアーケードゲーム「ブロック崩し」をWebブラウザで再現したものです。パドルを操作してボールを跳ね返し、画面上の全てのブロックを破壊することを目指します。

## 遊び方

-   **目的:** ボールをパドルで跳ね返し、画面上部のブロックを全て破壊すること。
-   **操作:**
    -   **パドル移動:** マウスを左右に動かすと、画面下部のパドルがそれに合わせて移動します。
    -   **ゲーム開始/ボール発射:** `スペースキー`を押すとゲームが開始し、ボールが発射されます。ゲーム中に`スペースキー`を押すと、ボールがパドルにくっついている状態から発射されます。
-   **ルール:**
    -   ボールが画面下部に落ちるとライフを1つ失います。
    -   ライフが全てなくなるとゲームオーバーです。
    -   全てのブロックを破壊するとゲームクリアです。
    -   ブロックを破壊するとスコアが加算されます。

## 特徴

-   **シンプルな操作:** マウスとスペースキーだけで簡単にプレイできます。
-   **カラフルなブロック:** 様々な色のブロックが登場し、視覚的に楽しめます。
-   **スコアとライフ:** 現在のスコアと残りライフが表示されます。
